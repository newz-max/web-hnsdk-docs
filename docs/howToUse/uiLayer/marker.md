# Marker 标记

<p>hnsdk.Marker 用于在地图上显示可点击/可拖动的图标。扩展自 <a href="#layer"><code>Layer</code></a>。</p>

<h2 id="marker-example">使用示例</h2>

```js
hnsdk.marker([50.5, 30.5]).addTo(map);
```
