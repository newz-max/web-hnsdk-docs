# 通过港口编码获取港口详细信息

通过港口编码获取港口详细信息

# 示例

async function (){
  const res = await hnSdk.queryPortDetail("CNQIN");
  console.log(res, "res");
}

# 参数

入参类型描述
mmsiListstring[]mmsi 数组

# 返回

Promise<{data : Response}>

# Response

选项类型描述
masterPortboolean是否为主港口
portQueryPortDetailResponsePort港口信息
portAttachmentQueryPortDetailResponsePortAttachment[]港口附件
portBerthsListQueryPortDetailResponsePortBerthsList[]港口泊位清单
portFacilityQueryPortDetailResponsePortFacility港口设施
portMediaQueryPortDetailResponsePortMedia[]港口媒体
portPlansQueryPortDetailResponsePortPlans[]港口计划
portSupplyQueryPortDetailResponsePortSupply[]港口供应
portsAuthAddressesQueryPortDetailResponsePortsAuthAddresses[]港口验证地址
portsOtherAddressesQueryPortDetailResponsePortsOtherAddresses[]港口其他地址
relatedPortsQueryPortDetailResponseRelatedPorts[]相关港口
